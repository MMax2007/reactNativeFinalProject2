В качестве проекта решил сделать инструмент для упрощения разработки обновлений игры, подключающийся к API, содержащей имеющиеся в игре элементы и предоставляющей возможность добавления новых элементов. Рассматривалась возможность встроить игру для тестирования добавляемых элементов, но времени не хватило.
